## A/B TEST

### 1. Overview

#### 🔸프로젝트 소개

본 프로젝트는 Kaggle의 두 가지 A/B 테스트 데이터셋을 활용하여 A/B 테스트의 개념, 분석 과정 및 실적 적용 사례를 하나의 흐름으로 정리한 분석 프로젝트입니다.

- 웹페이지 A/B Test Dataset : 사용자의 랜딩 페이지 변경이 전환율(Conversion)과 어떤 관계가 있는지 가설 검증 실습을 진행합니다.
- Cookie Cats Game Dataset : 모바일 게임 “Cookie Cats”에서 첫 번째 게이트 위치 변경이 유저 유지/리텐션에 미치는 영향을 검정합니다.

이 두 케이스를 통해 A/B 테스트의 실험 설계 → 통계적 가설 검정 → 결과 해석 및 의사결정 인사이트 도출 과정을 체계적으로 실습합니다.

#### 🔹프로젝트 목적

| 항목                                      | 설명                                                                            |
| ----------------------------------------- | ------------------------------------------------------------------------------- |
| 가설 검증 도구로서의 A/B 테스트 역할 정립 | 기능·UI 변경이 사용자 행동 지표에 미치는 영향을 통계쩍으로 검증하는 방법론 정립 |
